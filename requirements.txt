# ==========================================
# 1. 大语言模型与 RAG 核心框架 (LLM & RAG Core)
# ==========================================
langchain                # 核心框架：提供大模型编排、链式调用底层逻辑
langchain-openai         # 模型接口：用于接入兼容 OpenAI 协议的大语言模型（如 DeepSeek-V3）
#langchain-huggingface    # 词向量接口：用于在本地直接加载开源 Embedding 向量模型
langchain-community      # 生态扩展：提供对第三方组件（如 ChromaDB 向量库）的集成支持
langchain-text-splitters # 文本处理：提供高级文本切分算法，确保国标条文切片语义完整
openai                   # 基础网络层：处理底层 API 请求与网络重试机制

# 2. 向量数据库与多路检索 (VectorStore & Retrieval)
# ==========================================
chromadb                 # 向量数据库：轻量级本地持久化数据库，用于存储和高速检索文档 Chunk
#sentence-transformers    # 底层计算：本地加载模型计算余弦相似度的核心依赖
rank_bm25                # 稀疏检索算法：实现 TF-IDF 级别的关键词硬匹配，防止 RAG 漏召回关键国标编号
numpy                    # 数值计算：提供高效的向量运算支持，提升检索性能
# ==========================================
# 3. 文档解析与处理 (Document Parsing)
# ==========================================
pymupdf                  # 高性能 PDF 解析引擎：相比传统解析器能更精准地提取国标 PDF 文本结构

uvicorn                  # ASGI 服务器：作为底层服务器驱动 FastAPI 异步运行
pydantic                 # 数据验证引擎：通过 Python 类型提示进行严密的接口参数校验
pyinstaller 
PySide6

cryptography             # 安全加密库：解决 MySQL 8.0+ 默认的 caching_sha2_password 认证异常问题
